# Robotic-Arm
## Overview
This is a side project of mine in an attempt to create a 4 axis robotic arm. 
The arm utilized four Nema 17 motors on an open-loop feedback system attached
to an arduino uno with a CNC shield attachment running the AccelStepper Library.
